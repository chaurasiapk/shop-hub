## 🚀 Getting Started

1. **Repository Clone karo:**
   ```bash
   git clone https://github.com/chaurasiapk/shop-hub.git
   cd YOUR-REPO
   ```

2. **Dependencies install karo:**
   ```bash
   npm install
   # ya
   yarn install
   ```

3. **Development server start karo:**
   ```bash
   npm run dev
   # ya
   yarn dev
   ```

4. **Browser me open karo:**
   ```
   http://localhost:5173
   ```
